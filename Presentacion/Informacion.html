1
Andrea Nikolle Lozano Cortés
Socios Negocio
La primera función “CargarSocios” cumple el propósito de que, al ingresar al formulario se pueda visualizar la lista completa de los socios de negocios almacenados en la BD. Luego la función de “AgregarSocio” permite rellenar cada uno de los campos con información nueva, y posteriormente almacenarlos y así añadir un nuevo socio a la tabla. Posteriormente la función “CargarSocio” cumple con el objetivo de traer un socio especifico mediante el llamado del ID ayudando a continuación, a que la función “ActualizarSocio” puede realizar modificaciones en la información del respectivo socio y almacenarla nuevamente. Y por último la función “EliminarSocio” consiste en que mediate el llamado del ID del socio, borre el registro correspondiente tanto de la lista como de la BD. Cada una de las funciones mencionadas anteriormente, consume cada una de las 5 API Rest correspondientes realizadas en la primera parte del proyecto.

Estefany Gisell Pérez Rodríguez
Materiales
El formulario de "Materiales" en primera instancia muestra los datos de la tabla "Materiales" por medio de la función "CargarMateriales", además tiene otra función "AgregarMaterial" que permite agregar nuevos materiales a la base de datos, también brinda la posibilidad de hacer un llamado a un material en específico a través de la función "CargarMaterial", ya sea para editarlo o borrarlo, esto mediante las funciones de "ActualizarMaterial" y "EliminarMaterial".

Kristian Eduardo Elvir Blanco
Pedidos Proveedor
La funcion CargarPedidos muestra en pantalla los datos existentes almacenados en la base de datos, la funcion CargarPedido trae mediante el ID el registro especifico, la funcion AgregarPedido permite ingresar los datos del pedido y agregarlo a la base de datos, la funcion ActualizarPedido permite editar algun dato de los registros, la funcion EliminarPedido elimina el pedido mediante un ID especifico
4
Valeria Sophia Motiño Carias
Facturas Cliente
En el Formulario "Facturas Cliente" se tiene la primera función "CargarFacturas" la cual permite que se visualicen todos las Facturas de Clientes en la base de datos apenas se ingrese a el, como segunda función esta "CargarFactura" la cual obtiene todos los datos de una factura mediante un ID en especifico, en este caso el que se desea editar, como tercera función esta "AgregarFactura" la cual se encarga de ingresar una nueva factura al formulario y la base de datos, como cuarta función esta "ActualizarFactura" que nos permite cambiar y actualizar datos de las facturas que ya estaban en la base de datos, y como ultima función esta "EliminarFactura" que se encarga de eliminar una factura con todos sus registros de el formulario y la base de datos mediante un ID en especifico. Cada una de estas funciones consume su respectiva API Rest de las 5 que se realizaron en el proyecto anterior en facturas_cliente.php.
